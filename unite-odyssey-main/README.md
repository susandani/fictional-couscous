## Installation
1. Clone the repository:
```
git clone https://github.com/PrastianHD/unite-odyssey.git
```

2. Navigate into the project directory:
```
cd unite-odyssey
```

3. Install dependencies:
```
npm install
```

4. Configuration

- Go to path/src/proxy.txt
```bash shell
format proxy = `username:pass@host:port`
```

# Usage
```
npm start
```
